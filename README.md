# Advanced React Workshop for SeattleJS Conference

### To get started:

1. clone the repo   
```
git clone https://github.com/react-native-training/advanced-react.git
```

2. cd into directory   
```
cd advanced-react
```

3. install dependencies   
```
yarn
```
or
```
npm i
```

4. start the project   
```
npm start
```

### Dependencies used in this project:
React Router Dom   
React Motion   
Animated   
Glamor   
PropTypes   
React Primitives
